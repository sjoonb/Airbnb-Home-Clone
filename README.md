# Airbnb-Home-Clone

2021년도 2학기 두드림 프로젝트 결과물

## 프로젝트 설명

에어비앤비를 클론 코딩하는 프로젝트입니다. iOS 네이티브로 해당 앱으 구현하였으며, Firebase를 활용하여 기존 에어비앤비의 몇가지 대표적인 기능들을 축소 구현하였습니다. View 는 단순화 하였습니다.

## Firebase

Firebase 를 활용하여 기존 앱의 여러가지 기능을 구현하였습니다.
- 푸시알림 (firebase)
- auth 로그인 (firebase auth)
- 채팅 (fire base realtime-database)
- 프로필 (firebase auth)
- 호스팅: 데이터 읽기 및 쓰기 (fire store)
